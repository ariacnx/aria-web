# Model photos

Drop your photos here (`.jpg`, `.jpeg`, `.png`, or `.webp` — uppercase extensions like `.JPG` work too).  
They’ll show up on the **I model a bit** page automatically. No code changes needed.

**Tips:**
- **Mixed sizes are fine** — the gallery uses a masonry layout so portrait, landscape, and square all look good without cropping.
- **Resize before adding** — e.g. max 1200px on the long edge — for faster loading.
- **Filenames** become the caption (e.g. `sunset-melbourne.jpg` → “sunset melbourne”).
